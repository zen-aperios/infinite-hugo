# infinite-hugo

Infinite draggable tile wall for Webflow export.

## Files

- `index.html` runnable demo
- `style.css` demo styles
- `script.js` demo script
- `infinite-hugo.js` source script
- `infinite-hugo.min.js` minified script
- `infinite-hugo.css` source styles
- `infinite-hugo.min.css` min reference css
- `HOW_TO_WEBFLOW.md` embed guide

## Webflow CDN

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/zen-aperios/infinite-hugo@main/infinite-hugo.min.css" />
<script src="https://cdn.jsdelivr.net/npm/gsap@3.12.5/dist/gsap.min.js"></script>
<script src="https://cdn.jsdelivr.net/gh/zen-aperios/infinite-hugo@main/infinite-hugo.min.js"></script>
```
